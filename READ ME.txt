COMO CORRER O PROGRAMA:
Para correr o código basta fazer download dos nossos ficheiros estes incluem, 4 ficheiros python: agent.py, game.py, helper.py e model.py. A pasta deverá também conter um ficheiro com a fonte Arial. Para realizar este projeto usamos uma versão do python antiga, Python 3.7.9, é necessário instalar o pytorch(pip install torch torchvision torchaudio), o pygame(pip install pygame), o matplotlib(pip install matplotlib), e o IPython(pip install IPython), de seguida é necessário abrir a pasta que contém, os ficheiros do nosso trabalho num terminal, e usando a versão 3.7.9 podemos então correr o ficheiro python do agente(python agent.py).
